# Final Project - Emotion Detector

This repository contains the Final Project for the IBM Python for Data Science, AI & Development course.

## Project Name

Final Project - Emotion Detector

## Description

This project uses IBM Watson NLP to analyze text and detect emotions such as:

* Anger
* Disgust
* Fear
* Joy
* Sadness

The application is deployed using Flask and includes:

* Emotion detection package
* Unit testing
* Error handling
* Static code analysis using pylint
